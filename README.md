# Final Project for PRG 2104
## Description
This is a GUI application developed for the final project of PRG 2104 (April 2024 Semester).\
The project is built with the following:
- Scala 2.12.19
- Java SE 8
- ScalaFX 8.0
- SBT 1.9.7
